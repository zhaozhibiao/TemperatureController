# 温控仪硬件原理图与 PCB 设计实施指南

这份文档详细规划了基于选定 BOM 的 5 通道 Profinet 温控仪原理图设计模块及实施步骤。

## 1. 核心设计约束与物理架构

### 1.1 电源管理选型
* **主降压芯片**：已选用 **TI LMR16020** (60V, 2A 降压转换器)。
* **设计优势**：由于工业现场 24V 母线经常受到电机启停影响产生极高浪涌，选用 60V 耐压的 LMR16020 可提供极高的可靠性，无需担心普通降压芯片被击穿的风险。输入端需辅以 SMCJ33A TVS 二极管和自恢复保险丝实现完善的工业级防护。

### 1.2 导轨外壳与双板堆叠架构 (Double-layer Stacked Design)
为了适应紧凑的标准 DIN 导轨安装外壳（宽度受限），采用双层板堆叠设计：
* **底层板 (Base Board)**：主攻强电与对外接口。包含 24V 电源降压、双 RJ45 接口、5路 SSR 接口端子及热电偶端子。
* **顶层板 (Core Board)**：主攻弱电逻辑与模拟高精度采集。建议使用 4 层板走线。包含 STM32G4 主控、netX 90 芯片、ADS124S08 ADC、Type-C 接口等。
* **板间互联**：
  - **排母/排针 (Header)**：传输电源及低频控制信号。
  - **软排线 (FPC/FFC)**：用于传输高频以太网信号或跨板的传感器微弱模拟信号，提升装配灵活性并减少应力。

## 2. 硬件原理图模块划分

### 2.1 电源管理模块 (底层板)
* **输入保护**：24V DC 输入，加 SMCJ33A TVS 二极管。
* **主降压**：LMR16020 降压模块 (24V -> 5V)。
* **MCU 供电**：AMS1117-3.3 或 TLV70233。
* **隔离供电**：隔离型 DC-DC 电源模块（如 B0505S-1W）为 ADS124S08 和隔离器供电。

### 2.2 主控最小系统 (顶层板)
* **MCU**：STM32G474RET6。
* **时钟**：8MHz 外部高频晶振 (HSE) 用于精确的 USB 时序；32.768kHz 外部低频晶振 (LSE) 用于 RTC。

### 2.3 高精度隔离温度采集模块 (顶层板)
> **【警告】**：模拟地 (AGND) 与 数字地 (DGND) 必须在物理上完全隔离！
* **输入滤波与保护**：每一路差分对都需要 RC 低通滤波以及低漏电流的 ESD 保护阵列。
* **ADC模数转换**：使用 1 颗 TI ADS124S08 (24位)。
* **冷端补偿 (CJC)**：在 5个热电偶端子的物理中心位置，放置一颗 PT100 或高精度 NTC。
* **数字隔离**：STM32 到 ADC 的 SPI 总线必须经过 **ISO7741** 隔离芯片。

### 2.4 继电器控制输出模块 (底层板)
* **隔离驱动**：STM32 的 PWM 信号跨板到底层，驱动 **TLP281-4** 线性光耦。
* **状态指示**：在光耦的发光二极管侧串联一颗可见光 LED。

### 2.5 工业以太网通讯模块 (Profinet)
* **协处理器**：Hilscher netX 90 (顶层板)。
* **以太网口**：双 RJ45 接口，带内置网络变压器 (底层板)。

### 2.6 本地配置接口 (Type-C)
* **配置引脚**：CC1 和 CC2 必须分别通过 5.1kΩ 电阻独立接地。
* **数据线防护**：D+ 和 D- 必须加上专用 USB ESD 保护芯片（如 USBLC6-2SC6）。
